Usage is super simple:
1) Add ambient-addons to your inlude path
2) Import doc/*
3) Browse Page/Example.php for usage

Mind that internal limit for SpreadSheet is 250 cells, but you can
increse by extending SpreadSheet model and setting your own limit.

However, be careful, as depending on the field count in the model,
generation of a huge form takes some time and memory.
